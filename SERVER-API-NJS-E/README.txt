Estamos trabajando en un proyecto basado en NODEJS y EXPRES
REALIZAR un REST API
REST significa Representational State Transfer
La asignacion del servidor es un CRUD (Create, Read, Update, Delete) del Data.
El modulo HTTP tiene los siguientes methods: GET POST PUT DELETE
El ejercicio permitira visualizar clientes, un cliente especifico y cada metodo (GET, PUT, DELETE, POST)
En el archivo app.js levantamos un server con el modulo http sin express
Necesitamos instalar los siguientes modulos Express y Joi. 
Ademas en Chrome debes descargarte la app POSTMAN.
